对自增自减运算符的想法

根据运算符优先级来看：b = a++; 是先处理 a++ 再将其 返回的结果 赋值给 b

附表：Java运算符优先级

而自增的过程: (自减过程类似)

count++ : 先进行 +1 操作， 再将 原来的数值 返回;
++count : 先进行 +1 操作，然后将 变更后的结果 返回.

所以在上面代码中：

b = a++; 即为 a 增加 1, a = 1,再将原来的值 0 赋值给 b, b = 0;
a = a++; 即为 a 增加 1, a = 2,再将原来的值 1 赋值给 a, a = 1.
