基于 Markdown 的博文写作

发表于 2016-11-06 | 分类于 blog

Hexo 上支持用 Markdown 写一些简单的文章，但支持的写法较单一，如不能写流程图、脚注等。
好在对我而言这已经够用了，同时 Hexo 内置了一些标签插件。

在此我把一些常用的 Hexo 及 theme-nexT 的标签都写在这，方便查阅

  • Hexo 标签插件文档
  • theme-nexT 标签文档

changelog – 2017.3.3
最近发现另外几篇对 Markdown 博文写作整理的好文：

  • 基于Markdown的博文写作

Hexo 标签插件（Tag Plugins）

Image

在文章中插入指定大小的图片。

引用块

在文章中插入引言，可包含作者、来源和标题。
别号： quote

代码块

在文章中插入代码。
别名： code

反引号代码块

另一种形式的代码块，不同的是它使用三个反引号来包裹。

Include Code

插入 source 文件夹内的代码文件

Gist

在文章中嵌入 Gist

nexT 主题内置标签

经常换主题的情况下慎用！

文本居中的引用

此标签将生成一个带上下分割线的引用，同时引用内文本将自动居中。

突破容器宽度限制的图片

当使用此标签引用图片时，图片将自动扩大 26%，并突破文章容器的宽度。
